# Mbase Digital

Server-first marketing website for Mbase Digital.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Server-rendered/static content by default
- WebP-optimized project imagery

## Development

```bash
npm install
npm run dev
```

## Quality Gates

```bash
npm run lint
npm run build
```

## Brand Assets

Logo draft files are in `assets/brand/`. The website currently uses a temporary text mark until the final logo direction is approved.
